# Plant Database

> **Scope**: Timing tables, days-to-maturity, spacing, and zone suitability for common vegetables, herbs, fruits, and flowers.
> **Load when**: calculating indoor start dates, transplant dates, harvest windows, or spacing requirements.

---

## How to Use This Table

1. Find the user's last-frost date (from `seasonal-logic.md`).
2. For "Weeks before last frost" entries: count backward from last-frost date.
3. For "Weeks after last frost" entries: count forward from last-frost date.
4. "Direct sow" = no indoor start; sow directly in final container or bed.
